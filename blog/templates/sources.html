Sources

There are many sources from which I draw resources and components from this website; including images, concepts, and content. I will attempt to accomodate those sources here.

Django

Django is an powerful open-source web framework with a built-in relational database used to serve this very website.


Selenium Webdriver

Selenium Webdriver is a browser automation tool. It is useful in both creating test-cases for already existing browser applications, and test-driven development.


Fabric

Fabric is a Python library that designed to execute shell commands through SSH. This means instead of deploying manually, I use a pre-configured fabrication file to deploy automatically.


Gunicorn

Gunicorn (Gee-yoo-nee-korn) is a Python WSGI HTTP Server for UNIX, which forwards requests sent to the webserver to the django framework.


Digital Ocean

Digital Ocean is a cloud computing platform that provides virtual servers to businesses and individuals who do not have the infrastructure to maintain a server.


Django Markdownx

Django Markdownx is a plugin built for Django which allows my models to incorporate plain text formatting into my posts.
